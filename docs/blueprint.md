# **App Name**: PixelBlog SNES

## Core Features:

- Blog Posts: Displays a list of blog posts with titles, excerpts and categories.
- Category Filtering: Enables filtering of blog posts by category.
- Image Gallery: Displays a gallery of images in a grid layout with a lightbox.
- Sound Test: Emulates an SPC-700 audio chip for playing retro-style music tracks.
- Theme Customization: Generates customized website themes that match the tone and the style of the current content. A 'Customize Theme' button on the landing page allows you to use the generated themes.

## Style Guidelines:

- Primary color: Saturated pink (#FF77A8) for a retro and energetic feel.
- Background color: Dark navy (#1A1E4A) to provide contrast and highlight the content.
- Accent color: Electric blue (#6BE0FF) for interactive elements and highlights.
- Body and headline font: 'Press Start 2P' (monospace) for a pixelated, retro look.
- Alternate font: 'Inter' (sans-serif) for body text where readability is crucial.
- Use pixel-style icons to match the retro aesthetic.
- Implement a grid-based layout to organize content effectively.
- Add subtle animations, like fade-ins and slide-ins, for smoother transitions.
