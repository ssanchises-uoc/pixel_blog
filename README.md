# Proyecto para la asignatura: IA generativa para la creación multimedia 2025 UOC
### Autor: Santiago Sanchis Esparza

## PixelBlog - Edición SNES

Este es un blog con temática retro inspirado en la era de los 16 bits, construido con Next.js y estilizado con Tailwind CSS. 
- Repositorio de assets generados con IA
- Tutoriales técnicos enfocados en la creación de videojuegos 2D en pixel art estilo  16-bits (SNES/GBA).
- Sprites optimizados 
- Control de paletas
- Animaciones coherentes
- Buenas prácticas para spritesheets 
- Flujos de trabajo integrables en motores como Unity, Godot o GameMaker..
